# Module 3: Agents

Welcome to Module 3 of the Neo4j Generative AI workshop! This module teaches you how to build conversational agents that combine multiple retrievers into intelligent, tool-selecting systems.

## Module Overview

In this module, you will:

- Understand what agents are and how they wrap retrievers as conversational tools
- Learn how agents analyze queries and select appropriate tools automatically
- Build your first LangChain agent with schema introspection
- Progressively enhance your agent by adding Vector + Cypher and Text2Cypher tools
- Create a complete GraphRAG agent that intelligently chooses between multiple retrieval strategies
- Explore Neo4j Aura Agents for no-code agent creation
- Learn about LLM configuration and customization options
