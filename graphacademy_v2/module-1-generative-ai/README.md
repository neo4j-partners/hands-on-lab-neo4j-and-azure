# Module 1: Generative AI

Welcome to the Neo4j Generative AI workshop! This hands-on workshop will introduce you to the fundamental concepts of Generative AI and GraphRAG.

## Module Overview

In this module, you will:

- Recap key concepts about Generative AI and LLMs
- Understand LLM limitations including hallucination and data access issues
- Learn why context is crucial for accurate LLM responses
- Explore a knowledge graph of structured and unstructured data
- Learn about schema design for knowledge graphs
- Understand chunk size optimization for entity extraction
- Explore entity resolution techniques
- Search the graph using embeddings and vector indexes
